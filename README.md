# Linux Bootstrap

Bootstrap script for setting up a Linux machine with my public `rush` packages.

## Usage

Run this on a new Linux machine:

```bash
curl -Ls get.dannyb.co/bootstrap | bash
```

## Personal setup

If you are me, use `my-bootstrap` instead.

## Additional information

This script:

- Installs the [rush](https://github.com/DannyBen/rush-cli) package manager.
- Connects rush to my [rush-repo](https://github.com/DannyBen/rush-repo).
- Runs the public `bootstrap` package.
